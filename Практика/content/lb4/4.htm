4. ЛАБОРАТОРНАЯ РАБОТА.

ИССЛЕДОВАНИЕ ЗАВИСИМОСТИ «ВЕСА» РЕГУЛИРУЮЩЕГО СТЕРЖНЯ ОТ ЕГО ПОЛОЖЕНИЯ В АКТИВНОЙ ЗОНЕ И СКОРОСТИ ПОГРУЖЕНИЯ

 

4.1. Цель работы

Цель работы – изучение «веса» (эффективности) аварийной защиты (АЗ) и основных параметров, влияющих на его величину в различных состояниях активной зоны, изучение особенностей измерения эффективности АЗ на АЭС и интерпретации результатов измерений.

 

4.2. Задачи работы

Задачами данной работы являются:

1. Ознакомление с понятием «веса» АЗ, с его зависимостью от типа топливной загрузки, момента кампании и состояния активной зоны (по температуре и отравлению ксеноном), с особенностями измерения эффективности АЗ на АЭС.

2. Приобретение знаний об основных факторах, влияющих на эффективность АЗ, о методологии измерений эффективности АЗ и интерпретации результатов измерений.

3. Приобретение практических навыков и умения в проведении расчетных исследований восстановления эффективности АЗ на основе измерений и использования полномасштабных моделирующих комплексов, что необходимо для более ясного и точного представления о нейтронно-физических характеристиках ОР СУЗ.

4.4. Порядок выполнения работы

Лабораторная работа по данной теме состоит из трех этапов.

На первом этапе происходит исследование зависимости расчетной и измеряемой эффективности АЗ на МКУ для разных топливных загрузок от времени выдержки подкритического состояния активной зоны после сброса ОР СУЗ (сброс происходит мгновенно) и вида распределения источника фоновых нейтронов, построение временных зависимостей реактивности, вносимой погружением всех ОР СУЗ, и физическое объяснение этих зависимостей. Происходит изучение разницы между результатами статических и динамических расчетов балансной реактивности, разницы между величинами балансной реактивности и реактивности, регистрируемой по БИК, и определяется физическая причина этой разницы. На втором этапе работы происходит ознакомление с процедурой измерения веса АЗ и веса одного из ОР СУЗ для одного из указанных выше состояний активной зоны (или для одного и того же состояния зоны). Эта процедура выглядит следующим образом. Из управления выводится один из ОР СУЗ, который указывается на картограмме активной зоны на базовом формате данной лабораторной работы, а именно стержни с координатами: 13-22, 12-23, 11-26, 09-26. Сначала сбрасываются все ОР СУЗ без одного, фиксируется подкритичность реактора через 40–50 с, а затем добрасывается один застрявший ОР СУЗ и фиксируется стационарное значение подкритичности через 40–50 с при всех погруженных ОР СУЗ. Вес одного ОР СУЗ определяется как максимальная разница между подкритичностью без одного ОР СУЗ и подкритичностью при всех погруженных ОР СУЗ. При этом определяется и балансный вес ОР СУЗ тоже по разнице между значениями подкритичности реактора АЗ без одного ОР СУЗ и АЗ с добросом ОР СУЗ. В этом же моделировании сброса АЗ определяется вес одного ОР СУЗ из решения условно-критической задачи. Расположение отдельных ОР СУЗ выбрано таким образом, чтобы провести исследование зависимости измеренного и расчетного веса ОР от его удаления от ближайшей к данному стержню БИК. Ближайшей камерой является БИК с номером 3. Здесь же изучается зависимость измеряемого веса ОР от азимутального расположения камеры.

На третьем этапе работы происходит ознакомление с особенностями расчета и измерений эффективности АЗ при постепенном погружении ОР СУЗ в активную зону и изучение зависимости получаемых результатов от величины скорости погружения ОР СУЗ для одного из состояний активной зоны на МКУ зоны (или для одного и того же состояния зоны). Можно не разбивая обучаемых на 4 группы, поручить выполнение работы на 2-ом и на 3-ем этапах всем студентам для какого-либо одного и того же состояния активной зоны, к примеру для ВОС пусковой загрузки. Происходит построение интегральной и дифференциальной характеристик АЗ при погружении групп ОР СУЗ с минимальной скоростью (с шагом 0,05 от высоты извлечения), и проводится анализ рассогласования между измеренным и балансным весом.

На 1 этапе работа выполняется для всех указанных состояний, на 2-ом и 3-ем этапах работы – для одного из указанных состояний.

При выполнении лабораторной работы студенты разбиваются на 4 группы, которые выполняют следующие задания:

1 группа выполняет задания на 2-ом и 3-ем этапах для ВОС пусковой загрузки;

2 группа выполняет задания на 2-ом и 3-ем этапах для ЕОС пусковой загрузки;

3 группа выполняет задания на 2-ом и 3-ем этапах для ВОС для стационарной загрузки;

4 группа выполняет задания на 2-ом и 3-ем этапах для ЕОС для стационарной загрузки.

После выхода из программы посмотреть протокол можно через окно планировщика заданий (ПЗ). Появится окно протоколировщика, показанное на рисунке ниже.

 

 

Обработка протокола выполнения работы выполняется в следующей последовательности.

1. В меню «Графики» выбираем следующие переменные для отображения: SHOW_R(948) – признак для фильтрации; YSHGRP(…) – выбрать соответствующий работе номер (номера) группы ОР СУЗ, для АЗ любой номер; SHOW_R(551), SHOW_R(552) – вес группы на шаге и интеграл по шагам по БИК (%); SHOW_R(553), SHOW_R(554) – вес группы на шаге и интеграл по шагам по балансной реактивности (%).

Можно создать файл конфигурации графика, а при его наличии – загрузить файл конфигурации графика.

2. Отобразить на графике весь интервал времени.

3. Сохранить протокол. При сохранении желательно выбирать «говорящее» имя файла. Для этого нужно выбрать мышкой поле графика и нажать кнопку «Сохранение значений переменных в файл». Результаты моделирования сохраняются при этом в текстовом виде.

4. Фильтрация текстового файла с результатами моделирования выполняется из командной строки командой следующего вида:

filter_1.pl name.TXT > name_filter.TXT, где filter_1.pl – программа-фильтр, написанная на языке Perl, name.TXT и name_filter.TXT – имена файлов с результатами моделирования до и после фильтрации соответственно.

Для запуска программы фильтрации в директории с протоколами моделирования есть заготовка в виде bat-файла.

Для навигации по директориям тренажера, редактирования текстовых файлов и выполнения команд с командной строки удобно пользоваться файловым менеджером Far.

5. Редактируем файл name_filter.TXT в текстовом редакторе, выделяя и подписывая группы результатов для последующего построения графиков.

Выбор переменных для отображения протоколировщиком осуществляется в окне «Графики», показанном на рисунке ниже.

4.5. Содержание отчета

1. Графики зависимостей балансной и измеренной по БИК эффективностей ОР СУЗ после их сброса от времени на МКУ при единичном застревании различных ОР СУЗ.

2. Графики зависимостей балансной и измеренной по БИК эффективностей ОР СУЗ после их сброса от времени на МКУ при наличии фоновых нейтронов (эталонное моделирование) и графики без учета фоновых нейтронов.

3. Графики зависимости эффективности группы ОР СУЗ от числа шагов погружения ОР СУЗ.

4. Графики интегральных и дифференциальных характеристик по балансу и БИК ОР СУЗ на МКУ нейтронов при погружении ОР СУЗ за 20 шагов.

5. Результаты расчётов интегральных и дифференциальных характеристик групп ОР СУЗ в виде графиков по третьему этапу для шага 0,05; для балансной реактивности и для измеренной по БИК.
