1. ЛАБОРАТОРНАЯ РАБОТА.

ИНТЕРФЕЙС И УПРАВЛЕНИЕ АНАЛИТИЧЕСКИМ ТРЕНАЖЕРОМ ЭНЕРГОБЛОКА С РЕАКТОРОМ ВВЭР

 

1.1. Цель работы

Выполнение задания по изучению интерфейса аналитического тренажера энергоблока с реактором ВВЭР и возможностей по управлению.

 

1.2. Базовые понятия для работы с аналитическим тренажёром

Учебная лаборатория «Реакторная физика, управление и безопасная эксплуатация ЯЭУ» (аналитический тренажер) представляет собой интегрированный программно-технический и учебно-методический комплекс, имеет различные категории пользователей и обеспечивает возможность реализации ими следующих функций для обучающегося:

1. подготовка к лабораторным занятиям;

2. допуск к выполнению лабораторных работ;

3. выполнение лабораторных работ;

4. защита результатов выполнения лабораторных работ.

Для обеспечения реализации данных функций лаборатория укомплектована необходимыми средствами, а также руководствами по категориям пользователей.

Запуск готового задания. Для запуска нужного задания нужно выбрать учебный цикл (тему занятия) и соответствующий ей нужный каталог, войти в него и выбрать конкретную учебное задание для его запуска.

размещается ряд кнопок, которые позволяют управлять работой анализатора:

 – прекратить работу модели и выйти из оболочки GIW в планировщик заданий (ПЗ);

 – создание новых таблиц, графиков, форматов;

 – открытие уже готовых форматов, таблиц, графиков;

 – сохранение по старому месту или указанному вновь таблиц, графиков, форматов;

 – возврат к предыдущему виду экранных форматов;

 – возврат к текущему виду экранных форматов;

 и  – масштабирование участков форматов;

 – перемещение формата мышкой;

 – масштабирование формата целиком;

 – запуск модели на выполнение;

 – переход в режим редактирования форматов;

 – вызывает меню для сохранения нового состояния анализатора или считывания старого состояния. Также можно загружать значения переменных модели из текстовых файлов или сохранять их туда, можно включить аппаратное ускорение модели;

 – вызывает меню с именами всех загруженных форматов, таблиц и графиков;

 – вызывает меню, из которого можно вызвать список переменных модели, задавать отказы, заказать просмотр протокола, можно экспортировать различные форматы в метафайл для документирования и т.д.

В второй строке управляющей линейки программной оболочки GIW имеется ряд кнопок, которые доступны пользователю и полезны для редактирования форматов. Их функции кратко описаны во всплывающей строке при наведении стрелки «мыши» на каждую из этих кнопок, как и для всех остальных.

Запуск задания для выполнения лабораторной работы. Для начала необходимо выбрать модель-имитатора. Далее выбирается модель реактора.

 

 

Далее необходимо выбрать нужный лабораторный цикл.

 

Затем выбирается необходимая лабораторная работа, которая проводится на определённом занятии.

 

 

Далее выбирается задание для данной работы.

Действия по управлению заданием. Преподаватель указывает задание для выполнения 1-ой задачи. Правой кнопкой мыши преподаватель входит в меню для выбора вида запуска задания: удаленного выполнения или выполнения на собственном компьютере. Выбирается удаленный запуск и запускается задание. Инструкция преподавателя на этом исчерпывается и все остальные действия выполняются обучаемым на своем рабочем месте.

После загрузки задания появляется такое окно по управлению моделированием.

 

Далее вызывается состояние.

 

После загрузки состояния необходимо выполнять пошаговые инструкции, описанные в методических указаниях к каждой лабораторной работе.
