6. ЛАБОРАТОРНАЯ РАБОТА.

ИССЛЕДОВАНИЕ РЕЖИМА РАБОТЫ РЕАКТОРНОЙ УСТАНОВКИ ПРИ ОТКЛЮЧЕНИИ ОДНОГО ГЛАВНОГО ЦИРКУЛЯЦИОННОГО НАСОСА

 

6.1. Цель работы

Цель работы – изучение особенности протекания одного из режимов с нарушением нормальной эксплуатации, вызванного отключением одного из главных циркуляционных насосов; приобретение обучаемым знаний о работе систем безопасности в этом режиме, о корреляции между характеристиками систем управления и систем контроля, в данном случае автоматического контроля энергораспределения (АКЭ) и характеристиками объекта управления, об особенностях оперативного управления блоком в этой ситуации; приобретение практических навыков и умения в проведении расчетных исследований по выбору оптимальных действий по управлению энергоблоком.

 

6.2. Задачи работы

Задачами данной работы являются:

1. Исследование причин нарушения полевых ограничений по объему активной зоны при отключении ГЦН.

2. Исследование влияния работы системы АКЭ на процесс разгрузки энергоблока при отключении ГЦН.

3. Исследование по выбору нужной группы ОР СУЗ для реализации алгоритма ускоренной разгрузки блока (УРБ) при его использовании в режиме отключения одного ГЦН.

Первая группа запуска модели загружает третье состояние для реализации режима с отключением одного ГЦН, имевшего место в реальности, при одном распределении локальной мощности по зоне при работе РОМ с использованием алгоритма УРБ и вторым вариантом работы СГИУ и работой АКЭ при номинальной мощности 3120 МВт. Алгоритм действий при каждом запуске одинаков:

1. дождаться включения АРМ после разгрузки;

2. переключить АРМ на требуемую группу ОР СУЗ для УРБ;

3. после достижения высоты извлечения 8…20 % переключить АРМ обратно на 10 группу;

4. сразу включить подачу дистиллята в контур для компенсации отравления ксеноном;

5. маневрировать слегка группой ОР СУЗ в пределах указанного выше интервала.

Цель действий одна: не потерять оперативного управления блоком и не превысить полевые ограничения по локальной мощности в объеме активной зоны. Критерий извлечения группы ОР СУЗ для УРБ – увеличение запаса по KV. Как только запас по KV начинает убывать нужно переключать АРМ с группы для УРБ обратно на 10 группу.

В случае, когда максимальная неравномерность распределения поля энерговыделения находится в центре зоны, то же самое происходит с группой УРБ, на которую переключается АРМ. Переведите ускорение по размешиванию бора на 1. При повторном появлении тенденции по уменьшению запаса по КV можно повторить операцию по вводу чистой воды из бака TK70B01. Еще одна причина, по которой нужно разбавлять воду в 1-ом контуре дистиллятом – это уменьшение мощности из-за ксенонового отравления активной зоны, но данный процесс очень медленный и хорошо контролируемый. Нужно дождаться стабилизации запаса по KV, появления надписи во втором окне в верхней части базового формата о завершении задания и завершить лабораторную работу.

 

Обработка результатов работы.

Результаты протоколирования размещаются в выходном файле с относительный маршрутом:

.,,\VVER1000\MODELS\USER’ NAME_MFA_RD\PROTOCOL\ \LAB_04.1\NAME OUTPUT FILE.

Имя выходного файла устанавливает сама система по имени задания, включая все вложенные каталоги и время выполнения.

Из протоколируемых переменных нужно выделить следующие величины:

1. тепловая мощность по СВРК – YMINTPOWMWT;

2. тепловая мощность по БИК – YCINTCAMMWT;

3. Электрическая мощность – N_TURB_RL;

4. Положение управляющих групп 9, 10 – YSHGRP(10),YSHGRP(9);

5. Запас по KV – YZ_DKV_MIN;

6. Запас по Ql – YSVRK_DMIN12(1);

7. Уставка РОМ – ROM01C2;

8. Параметры перехвата движения групп ОР СУЗ – yslimswitch80, yslimswitch20;

9. Максимальный коэффициент неравномерности локальной мощности ТВС по зоне KQmax – YM_KQ_W_MAXn;

10. Максимальный коэффициент неравномерности энерговыделения по объему зоны KМmax – YM_KV_W_MAXn.

6. Графики зависимости величин KQmax и KVmax и положения групп ОР СУЗ от времени для наихудшего варианта.

7. Заключение с анализом результатов расчетных исследований и с определением наилучшего варианта для разгрузки блока при отключении одного ГЦН; с рекомендациями по совершенствованию алгоритма УРБ в системе управления реактором в зависимости от распределения нейтронной полей по активной зоне.
